This is my npc generator I created for DnD. It is my first project ive done since quitting school.

The goal is to generate a random name, positive and negative trait, and a negotiation/motivation from a list.
I used .txt files for the lists as i wanted to be able to easily edit the names the generator pulled from. As well as in the actual code I made it so that you simply have to copy one of the created functions to add
additional races to the generator.

In order to add another race te be able to generate, simply add the first and last name .txt files with the list of names you wish to randomize from into the folder, add the ability to open the files in the file open statement.
And copy a function, change all the named sections to the race you are adding, and it should work.
